<ol> Tag

Description:

The HTML ordered list element represents an ordered list of items with a numerical ordering. Typically, ordered-list items are displayed with numbers, letters, or even Roman Numerals. One can specify a start

Example:

<ol start="3" type="i">
    <li>Water</li>
    <li>Milkshake</li>
    <li>Coffee</li>
    <li>Smoothie</li>
</ol>

Result:

  1. Water
  2. Milkshake
  3. Coffee
  4. Smoothie
Try it yourself!
