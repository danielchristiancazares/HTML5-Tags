<video> Tag

Description:

Denotes a video.

Example:

<video width="360" height="240" controls>
    <source src="assets/SampleVideo_360x240_1mb.mp4" type="video/mp4">
</video>

Result:

Try it yourself!
Tag-Specific Attributes
Tag Input Values Description
autoplay N/A Specifies that the video will start playing as soon as it is ready
controls N/A Specifies that video controls should be displayed (such as play, pause, and mute).
height pixels Sets the height of the video player
loop N/A Specifies that the video will start over again every time it is finished
muted N/A Specifies that the video will be muted
poster URL Specifies an image to shown while the video is downloading, or until the user hits the play button
preload auto
metadata
none		 Specifies if and how the author thinks the video should be loaded when the page loads
src URL Specifies the URL of the video file
width pixels Sets the width of the video player
