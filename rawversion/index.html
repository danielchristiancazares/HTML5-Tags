Welcome to our HTML reference page.

Search for an HTML5 tag in the top-right corner, or find it below.

Tag Description
<a> Defines a hyperlink that will take the user to a new page.
<abbr> Defines an abbreviation. Used to give information to browsers and search engines.
<address> Specifies author's contact address (electronic or mailing).
<area> Defines a specific area inside an image-map. Must be inside a <map&rt; tag
<article> Defines an article in the document.
<aside> Defines content aside from the page content.
<audio> Defines sound content.
<b> Defines bold text.
<base> Defines the base URL for all relative URLs in a page.
<bdi> Defines a part of text that might be formatted in a different direction from the other text.
<bdo> Defines right-to-left or left-to-right text direction.
<blockquote> Specifies a section of text quoted from another source \.
<body> Defines the document's body.
<br> Defines a line break.
<button> Defines a clickable button.
<canvas> Creates a space on which scripts can draw objects .
<caption> Defines a table's caption.
<cite> Declars the title of an external piece of work.
<code> Defines a piece of computer code.
<col> Specifies properties for columns.
<colgroup> Specifies a group of columns in a table.
<data> Allows for machine-readable data to be provided .
<datalist> Specifies an "autocomplete" dropdown list .
<dd> Defines a description of a term in a description list.
<del> Defines text that has been deleted from a document .
<details> Specifies details of an element.
<dfn> Specifies the defining instance of a term.
<dialog> Specifies that part of an application is interactive .
<div> Defines a division or section within a doc. Usually for CSS. .
<dl> Defines a description list.
<dt> Defines a term in a description list.
<em> Defines emphasized text.
<embed> Specifies external application or interactive content .
<fieldset> Defines a group of related elements in a form.
<figcaption> Specifies caption for the figure element. .
<figure> Specifies a group of media content, and their caption .
<footer> Specifies a footer for a section or page .
<form> Defines a form for user input.
<h1> Headings of varying sizes. h1 (largest) to h6 (smallest).
<head> Defines information about the document.
<header> Specifies a group of introductory or navigational aids, including hgroup elements .
<hgroup> Specifies a header for a section or page. .
<hr> Inserts a line across the page, usually to indicate a change in content.
<html> Defines the root of the HTML document.
<i> Defines text of a differtent voice or mood.
<iframe> Defines an inline frame.
<img> Defines an image.
<ins> Defines inserted text.
<kbd> Defines keyboard input.
<keygen> Generates a key pair.
<label> Defines the label for an input element.
<legend> Caption for a fieldset.
<li> Defines a term in a list.
<link> Defines a link between this document and an external resource.
<main> Specifies the main content area of an HTML document .
<map> Defines an image-map.
<mark> Specifies marked text.
<meta> Defines metadata for the document; not displayed .
<meter> Specifies measurement within a predefined range .
<nav> Specifies navigation links .
<object> Defines an embedded object.
<ol> Defines an ordered list.
<optgroup> Defines options that can be grouped together in a drop-down list.
<option> Defines an option in a drop-down list.
<output> Defines the result of a calculation.
<p> Defines a paragraph for text.
<param> Defines a parameter for an object.
<pre> Defines preformatted text.
<progress> Defines the progress of a task.
<q> Defines a quotation.
<rb> Defines the base of a ruby annotation.
<rp> Specifies text to show for browsers that do not support ruby annotations.
<rt> Defines annotated ruby text.
<rtc> Defines a text container for ruby annotations.
<ruby> Defines a segment of text for ruby annotations.
<s> Specifies strikethrough text.
<samp> Defines a sample output from a program.
<section> Defines a section in a document.
<select> Defines a drop-down list.
<small> Specifies small text.
<source> Defines resources for media elements.
<span> Defines a section in a document.
<strong> Defines important text.
<style> Defines style information, usually used for CSS.
<sub> Specifies esubscripted text.
<summary> Defines a heading for a <details> element.
<sup> Specifies superscripted text.
<table> Defines a table element.
<tbody> Specifies the body content of a table.
<td> Defines table data that will appear in a cell in the table.
<template> Declares HTML fragments that can be cloned and inserted in the document by script.
<tfoot> Specifies footer content for a table.
<th> Specifies a header cell in a table.
<thead> Groups the header content in a table.
<time> Specifies a date/time.
<title> Defines the title for the document.
<tr> Specifies a row in the table.
<track> Specifies a text track for media such as video and audio.
<u> Specifies underlined text.
<ul> Defines an unordered list.
<var> Defines a variable.
<video> Specifies a video.
<wbr> Specifies a line break opportunity for very long words and strings of text with no spaces.
