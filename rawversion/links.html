Link Tags

Tag Description
<a> Defines a hyperlink that will take the user to a new page.
<area> Defines a specific area inside an imae-map. Must be inside a <map> tag
<base> Defines the base URL for all relative URLs in a page.
<link> Defines a link between this document and an external resource.
<nav> Specifies navigation links .
<source> Defines resources for media elements.
