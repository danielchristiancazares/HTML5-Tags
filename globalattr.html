Global Attributes

Attribute Values (if necessary) Description
accesskey keyboard key (e.g. "a" or "c") Specifies a shortcut key to activate/focus a tag. (Depends on browser, but usually Alt + accesskey is the way to use the accesskey)
class classname(s) (user-defined) Specifies one or more class names for an element which will be subject to changes from the referenced style sheet.
contenteditable true
false		 Specifies whether the content of the element is editable.
contextmenu menu_id (user-defined) Specifies a context menu (given by menu_id) for an element. The menu appears upon right-clicking the element.
data-* somedata (user-defined) Used to store custom data private to the page or app.
dir ltr
rtl
auto		 Specifies the text direction of an element's contents. ltr is left to right, and rtl is right to left. auto lets the browser figure out the text direction.
draggable true
false
Specifies whether an element is draggable. auto uses the default behavior of the browser.
dropzone copy
move
link		 Specifies whether the dragged data when dropped is copied, moved, or linked <./td>
hidden N/A Specifies that the element is not visible.
id idname (user-defined) Specifies a unique id for the element.
lang language_code Specifies the language code for the element's content.
spellcheck true
false		 Specifies whether element should have its spelling and grammar checked.
style style_definitions (user-defined, in accordance to CSS) Specifies an inline CSS style for an element.
tabindex number Specifies the tabbing order of an element.
title text (user-defined) Specifies extra information about an element.
translate true
false		 Specifies whether the content should be translated

Note: User-defined means that the web developer defines these values.
